RimWorld Beginner's Guide & Cheat Sheet

From a 4,000+ hour veteran player

Game Start

Starting Your Colony

Scenario Selection

  • Crashlanded (Recommended): 3 colonists, industrial tech, balanced start
  • Lost Tribe: 5 colonists, primitive tech, slower research
  • Rich Explorer: 1 colonist, advanced tech, lots of resources
  • Naked Brutality: 1 colonist, no resources (hardest)

Storyteller Choice

  • Phoebe Chillax: Easiest, relaxed pacing
  • Cassandra Classic: Consistent difficulty curve, predictable raids
  • Randy Random: Unpredictable events, can be very hard

Map Selection

  • Biome: Pick Temperate Forest for beginners
  • Terrain: Small Hills recommended (balance of space and resources)
  • Growing Period: 40-60 days minimum, or year-round (60/60)

Character Selection & Initial Setup

Character Selection Rules

  • Each skill should have at least one colonist with a passion (flame icon)
  • Avoid colonists who can't do violence
  • Avoid colonists over 50 (health issues)
  • Key skills needed: Medical, Cooking, Construction, Plants, Mining
Passion System:
  • No passion = 35% learning speed
  • Minor passion (1 flame) = 100% learning speed
  • Major passion (2 flames) = 150% learning speed

First Actions (In Order)

  1. Pause game (Spacebar)
  2. Unforbid all items (Architect → Orders → Allow All)
  3. Set work priorities (Work tab → Manual priorities)
  4. Plan your base location (13x13 tiles)
  5. Create stockpile zone inside base area
  6. Create growing zone on fertile soil
  7. Equip weapons and armor

Work Priority Setup

Task Priority Notes
Firefight 1 Always highest priority
Patient/Doctor 1 Critical for survival
Basic/Warden 1 For switches, prisoners
Cook/Hunt 2 Food production
Construction 2 Base building
Growing/Mining 3 Resource gathering
Crafting/Art 3 Equipment and mood

Base Building Guide

Initial Base (Day 1)

  • Build 13x13 wooden walls with door
  • Place stockpile zone inside
  • Add 3 beds before nightfall
  • Build horseshoe pin for recreation
  • Add torch lamp for light
  • Create growing zone (90 tiles of rice)

Essential Buildings (Days 2-5)

  • Research Bench: Start with Batteries research
  • Table & Chairs: Prevents "ate without table" mood debuff
  • Butcher Table: Process hunted animals
  • Stone Cutting Table: Make stone blocks for fireproof walls

Power & Temperature

  • Wind Turbine: Place over crops (no trees to block)
  • Solar Panels: Consistent daytime power
  • Batteries: Store excess power
  • Freezer Setup:
    • Small room next to base
    • Cooler with blue side facing in
    • Set to -5°C
    • Don't roof the exhaust area
Wall Upgrade Path: Wood → Stone (marble/granite) → Steel (avoid - flammable!)

Colony Management

Food Management

  • Rice: Fastest growing, labor intensive
  • Corn: Slow growing, less labor
  • Healroot: Essential for medicine
  • Cotton: For clothing production
Cook simple meals to avoid mood debuffs. Set bill to "Do until you have 10" with "Pause when satisfied" at 4.

Mood Management

  • Individual bedrooms (eventually)
  • Impressive dining/rec room
  • Art sculptures in common areas
  • Comfortable furniture
  • Clean environment
  • Variety in recreation

Research Priority

  1. Batteries
  2. Solar Panels
  3. Smithing
  4. Machining
  5. Microelectronics
  6. Gun Smithing (for weapons)

Combat & Defense

Defense Setup

  • Spike Traps: Place in corridors leading to base
  • Doors Between Traps: Colonists avoid triggering them
  • Barricades: Cover for shooters
  • Choke Points: Force enemies into narrow spaces

Combat Tactics

  • Kiting: Shoot and retreat against melee enemies
  • Focus Fire: All colonists target same enemy
  • Draft Mode: Press 'R' for manual control
  • Downing Raiders: Use blunt weapons to recruit

Prisoner Management

  • Build separate prisoner room
  • Set bed to "For Prisoners"
  • Assign warden with high Social skill
  • Choose: Recruit, Release, or Harvest Organs

Pro Tips & Shortcuts

Keyboard Shortcuts

  • Space - Pause/Unpause
  • 1, 2, 3 - Game speed
  • R - Draft/Undraft
  • B - Attack command
  • Q/E - Rotate objects
  • Shift + Click - Queue orders

Advanced Tips

  • Animals can haul - train them!
  • Shelves stack more items in less space
  • Organs sell for huge profits
  • Stone chunks provide 20 blocks each
  • Set bills to "Drop on floor" to save hauling time
  • Home zone determines cleaning/repair area
  • Double-click to select all of same type

Common Mistakes to Avoid

  • Building with steel (it's flammable!)
  • Forgetting to set cooler temperature
  • Not managing clothing quality (51% minimum)
  • Ignoring mood management
  • Fighting in the open instead of choke points
  • Not having enough medicine stockpiled

Remember: Every colony has a story. Embrace the chaos!
